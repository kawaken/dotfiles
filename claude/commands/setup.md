# Setup dotfiles

このリポジトリのセットアップを行う。

## 手順

1. `README.md` の `## install` セクションを読む
2. 現在の環境（`$HOME`、既存のシンボリックリンクや設定ファイル）を確認する
3. 未適用の手順をリストアップしてユーザーに提示する
4. ユーザーの確認を得てから実行する

## 注意事項

- 既に適用済みの手順はスキップする
- `user.name` / `user.email` など個人情報が必要な手順はユーザーに入力を求める
- `~/.claude/settings.json` は `claude/settings.base.json` をベースにする。新規ならコピー、既存なら `jq -s '.[0] * .[1]'` でマージ（ベース側優先）
- 既存ファイルを上書き・削除する操作は必ず確認を取ること
