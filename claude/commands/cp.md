# Commit and Push

変更をコミットしてプッシュする。

## 手順

1. `git status` で変更を確認
2. 変更がなければ終了
3. 変更があるファイルをユーザーに提示し、どのファイルをコミットするか確認する
4. 確認後は以下を一気に実行（途中確認不要）：
   - 選択されたファイルを `git add` でステージング
   - `git diff --cached` で差分を確認し、変更内容に応じたコミットメッセージを生成
   - `git commit` でコミット（Co-Authored-By は不要）
   - `git push` でプッシュ

## コミットメッセージ

- 簡潔でOK
- 変更内容を端的に表すメッセージにする
